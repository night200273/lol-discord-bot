## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Set Discord Bot Token:
```bash
set DISCORD_TOKEN=your_bot_token_here
```

3. Run the bot:
```bash
python main.py
```

## Deployment on Heroku

This bot is ready to deploy on Heroku with the included `Procfile`.

## Requirements

- Python 3.8+
- discord.py
